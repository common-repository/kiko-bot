== Description ==

Ein Kiko Chatbot beantwortet dem Besucher der Website jederzeit die häufig gestellten Fragen. Der Chat kann als Widget oder als Teil einer Webseite eingebunden werden.

== Information ==

Mit wenigen Schritten können Sie sich für ein kostenfreies Chatbot-Konto (Basis-Plan) registrieren.

Bei der Registrierung wird ein eigener Chatbot angelegt. Unter [support.kiko.bot](https://support.kiko.bot) erhalten Sie wichtige Informationen zur Inhaltspflege.

== Installation ==

= Installation Instructions =

* Installieren Sie das Plugin "Kiko Chatbot" über das "Plugin" Menü.
* Registrieren Sie sich mit wenigen Klicks auf [www.kiko.bot](https://www.kiko.bot) oder tragen Sie den API-Key Ihres Kiko-Kontos im Plugin ein.
* Anschließend können Sie den Chatbot-Inhalt auf [www.kiko.bot](https://www.kiko.bot) pflegen und veröffentlichen.
* Per Plugin steht der Chatbots als Widget oder als Teil einer Wordpress-Webseite per Shortcode zur Verfügung.
